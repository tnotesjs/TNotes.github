## 3. 🤔 Musicxx 是什么？

Musicxx 是一款采用新拟物风格（Neumorphism）设计的音视频播放器开源项目，旨在为用户提供美观且实用的跨平台播放体验。

界面 UI 截图：

![2](https://cdn.jsdelivr.net/gh/tnotesjs/imgs@main/2025-07-01-07-53-04.png)

:::

主要特点：

| 特性       | 说明                               |
| ---------- | ---------------------------------- |
| 跨平台支持 | Android、Windows、macOS、Linux     |
| 界面风格   | 新拟物风格（Neumorphism）设计      |
| 多端同步   | 支持账号多端同步，无缝衔接听歌体验 |
| 格式支持   | 支持多种音乐和视频格式             |
| 开源免费   | 完全开源，可自由使用和二次开发     |

技术架构：

- 后端: 使用 C++ 开发，基于搜狗开源的 workflow 框架
  - 支持网络开发和计算任务调度
  - 包含 HTTP、RPC、MySQL 等功能模块
- 客户端: 支持多种客户端实现（包括 Flutter 等）

适用场景：

- 个人音乐/视频播放需求
- 跨平台媒体播放解决方案
- 学习 C++ 后端开发和跨平台应用开发
- 新拟物风格 UI 设计参考
