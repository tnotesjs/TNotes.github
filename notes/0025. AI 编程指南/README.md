## 2. 评价

这一部分更多会记录一些类似 prompt 相关的内容，让 LLM 学习给定的提示规范之后再按照规范进行编程。

主要记录了一些 SKILLS，在开发阶段集成到项目中，让 Agent 遵循 SKILLS 的规范进行编程，有可能可以提高编程效率，少几轮对话就能完成任务，减少一些不必要的对话成本。
