## 1. 🎯 本节内容

- REKCARC-TSC-UHT

## 2. 🫧 评价

这个开源项目记录了清华计算机系大一到大四以及研究生的相关课程以及一些学习资料。

## 3. 🤔 这个仓库中都包含哪些核心内容？

项目中的核心内容包括：

| 内容类别 | 具体内容             | 描述                         |
| -------- | -------------------- | ---------------------------- |
| 课程材料 | 按学期分类的课程指南 | 各学期的课程学习指导和攻略   |
|          | 编程作业             | 课程相关的编程练习和作业     |
|          | 实验报告             | 实验课程的报告模板和范例     |
| 考试资源 | 历年试卷             | 各课程的期末、期中考试试卷   |
|          | 试题解答             | 考试题目的参考答案和解题思路 |
| 参考材料 | 参考书目             | 课程相关的书籍和资料推荐列表 |
| 文档     | README.md            | 项目主要说明文档             |
|          | 收录内容.md          | 仓库收录内容的详细索引       |
|          | 贡献方法.md          | 如何参与项目贡献的指南       |

具体有哪些内容，可自行打开项目查阅。
