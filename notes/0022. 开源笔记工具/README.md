## 2. 评价

这一部分记录和“笔记”相关的一些开源工具，包括：

- 开源的画板工具
  - excalidraw：截止目前（26.06）已经用很久了，[TNotes.leetcode][1] 这个知识库中写的算法图解基本都是用这个 excalidraw 来绘制的
  - tldraw：和 excalidraw 类似，也是一个开源画板工具
- 开源的 PDF 解析工具
  - MinerU：很好用，目前（26.06）解析 PDF 主要用的就是它
- 开源的笔记软件
  - memorains：B 站上刷到的一位 UP “红鳞绿羽”做的一款开源笔记软件
- 开发者使用的工具
  - Quill：富文本编辑器
  - YJS：共享数据，协同编辑
